# Controller
GET 　/home  -> 掲示板画面
GET 　/login -> ログイン画面

GET 　/user/register -> ユーザー登録画面
POST  /user/register -> ユーザー登録実施

# RestController
GET     /api/message      -> メッセージ取得
POST    /api/message      -> メッセージ投稿
DELETE  /api/message/{id} -> メッセージ削除

POST    /api/good/{id}    -> いいね投稿
DELETE  /api/good/{id}    -> いいね削除

# Security
POST    /security/login  -> ログイン実施
POST    /security/logout -> ログアウト実施
